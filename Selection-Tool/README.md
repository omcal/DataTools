------------
#### Steps to  build project 
Requirements:
Node and nvm should be installed
- Node version should be 16.10.0
---------------------------------
- `nvm install 16.10.0`
- `clone the project`
- `nvm use 16.10.0`
- `yarn install` it will take a little bit time
- `yarn start` you can start the project
